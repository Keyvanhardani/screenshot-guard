# screenshot-guard

Secret scanner with OCR superpowers - finds secrets in code AND screenshots.

## Installation

```bash
npm install -g screenshot-guard
```

This will also install the Python package automatically.

## Usage

```bash
screenshot-guard scan .
screenshot-guard scan ./src --format json
screenshot-guard scan . --severity high
```

## Requirements

- Node.js >= 16
- Python >= 3.10
- pip
